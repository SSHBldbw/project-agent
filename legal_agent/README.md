# ⚖️ 法智AI - 法律助手智能体

基于 RAG 技术的专业法律智能助手，提供案件分类、风险评估、文书生成等一站式法律服务。

## ✨ 核心功能

### 📋 案件类型智能分类
根据案件描述自动识别案件类型（民事/刑事/行政），提供：
- 案由分类
- 适用法律条款
- 管辖法院建议
- 举证责任说明
- 诉讼时效提示

### ⚠️ 法律风险评估
智能识别法律场景中的潜在风险：
- 风险等级评定（高/中/低）
- 潜在后果分析
- 综合风险建议

### 📝 法律文书模板生成
支持多种标准法律文书模板：
- 民事起诉状
- 离婚协议书
- 借款合同/借条
- 劳动合同
- 房屋租赁合同
- 授权委托书

### 📚 法典检索
基于向量+关键词混合召回 + CrossEncoder精排，支持：
- 《中华人民共和国民法典》
- 《中华人民共和国刑法》

### 🔍 联网搜索
通过百度搜索获取最新法律法规动态和司法解释。

## 🚀 快速开始

### 环境要求
- Python 3.10+
- 依赖库：见 `requirements.txt`

### 安装步骤
```bash
cd d:\AI_\test\legal_agent
pip install -r requirements.txt
```

### 配置文件

在 `d:\AI_\.env` 中配置以下环境变量：
```ini
# 大模型配置
DASHSCOPE_API_KEY=your-dashscope-key
LLM_MODEL=qwen3-max

# 搜索配置
BAIDU_API_KEY=your-baidu-api-key

# 嵌入模型配置
EMBEDDING_MODEL=BAAI/bge-large-zh-v1.5
EMBEDDING_BASE_URL=https://api.siliconflow.cn/v1
EMBEDDING_API_KEY=your-siliconflow-key

# 向量库配置
LAW_CHROMA_PERSIST_DIR=.chroma_legal
```

### 启动服务

**启动前端界面：**
```bash
streamlit run app.py --server.port 8501
```

访问 http://localhost:8501 即可使用。

## 📌 使用示例

### 案件分类
```
输入：朋友借了我10万块钱不还，还把我微信拉黑了
输出：民事案件 → 民间借贷纠纷，适用民法典合同编
```

### 风险评估
```
输入：我和朋友合伙开店，只有口头协议
输出：高风险 - 无书面合同风险，举证困难
```

### 文书生成
```
输入：帮我生成一份离婚协议书模板
输出：标准离婚协议书结构 + 注意事项
```

## 📖 法典文件格式

法典文件需为 `.txt` 格式，文件名包含"中华"字样，编码支持 GBK/GB18030/UTF-8。
